# Помощник на Яндекс Cloud Functions

Тёплый ИИ-помощник клуба работает так: браузер шлёт запрос в **функцию Яндекс
Cloud**, а она обращается к **YandexGPT** от имени сервисного аккаунта. Ключ
YandexGPT в браузер не попадает.

## Что нужно (у Елены уже есть)

- Каталог (folder): `b1gq9mjq2s1jkmikmun6`
- Сервисный аккаунт с ролью `ai.languageModels.user`

## Как создать функцию (с айпада, в консоли Yandex Cloud)

1. **Cloud Functions → Создать функцию.** Имя: `assistant`. Создать.
2. Вкладка **Редактор**:
   - Среда выполнения: **Node.js 18** (или новее).
   - Способ: **Редактор кода**.
   - Файл `index.js` — вставить содержимое `yandex/assistant/index.js`.
   - **Точка входа:** `index.handler`.
3. **Параметры** (на этой же странице, справа/ниже):
   - **Сервисный аккаунт:** выбрать тот, у кого роль `ai.languageModels.user`.
     (Тогда ключ вообще не нужен — функция берёт токен аккаунта сама.)
   - **Переменные окружения:** добавить `YANDEX_FOLDER_ID` = `b1gq9mjq2s1jkmikmun6`.
   - Таймаут: 30 сек, памяти хватит 128 МБ.
4. Нажать **Создать версию**.
5. Вкладка функции **Обзор → Сделать публичной** (публичный вызов). Появится
   адрес вида `https://functions.yandexcloud.net/d4e...`.
6. **Скопировать этот адрес и прислать мне.** Я подставлю его в приложение
   (`src/lib/assistant.ts`) и залью — после этого помощник заработает.

## Если сервисного аккаунта нет

Можно вместо него добавить переменную окружения `YANDEX_API_KEY` со значением
ключа. Но привязать сервисный аккаунт надёжнее и проще — тогда ключ нигде не
хранится.
